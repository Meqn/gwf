
# 自定义配置

> 在项目下新增自定义配置文件 `.build.config.json`


## demo

文件 `.build.config.json`

```json
{
  "host": "",
  "public": "",
  "server": {
    "port": 8080,
    "proxy": []
  },
  "output": {
    "hash": true,
    "cssCombo": true,
    "jsCombo": true,
    "compressJs": true,
    "compressCss": true,
    "compressImage": true,
  }
}
```
